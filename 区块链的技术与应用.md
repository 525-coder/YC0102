# 区块链的技术与应用

## 比特币系统中用到的密码学原理

1. 哈希函数 cryptographic hash function 

   ​	性质1：collision resistance （抗碰撞性）

   ​				是指不能轻易找出x不等于y，而H（x）=H（y）。

   ​	性质2：hiding 

   ​				hash函数的计算过程是单向的不可逆的。成立的前提是输入空间足够大，取值比较均匀

   ​				可以和抗碰撞性应用为数字签名。

   ​	性质3：puzzle friendly

   ​				使得哈希值计算事先不可与预测
